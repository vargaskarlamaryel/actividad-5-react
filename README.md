# Actividad 5 - React

## Descripción

Proyecto React con Vite que demuestra:

- Imagen cargada desde `public`.
- Imagen importada desde `src/assets`.
- Select dependiente de otro select.
- Manejo de estado con `useState`.

## Cómo ejecutar

```bash
npm install
npm run dev
```

## Cómo desplegar en Vercel

1. Subir el proyecto a un repositorio público de GitHub.
2. Entrar a Vercel.
3. Seleccionar **Add New Project**.
4. Importar el repositorio.
5. Presionar **Deploy**.
6. Copiar la URL pública.

## Entrega requerida

- Enlace del repositorio.
- URL pública del despliegue.
